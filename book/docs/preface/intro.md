# Preface & How to Use This Book

Welcome to **Physical AI & Humanoid Robotics**, a comprehensive textbook designed to guide you through the exciting world of physical artificial intelligence and humanoid robot development.

## About This Book

This textbook is structured as a 13-week course covering:

1. **Foundations** (Weeks 1–2): Core concepts of Physical AI
2. **ROS 2 Nervous System** (Weeks 3–5): Robot Operating System fundamentals
3. **Digital Twin** (Weeks 6–7): Simulation with Gazebo and Unity
4. **NVIDIA Isaac Platform** (Weeks 8–10): Advanced robotics platform
5. **Humanoid Development** (Weeks 11–12): Building humanoid robots
6. **Conversational Robotics** (Week 13): Voice and language interaction

## How to Use This Book

### For Students

- **Read sequentially**: Each chapter builds on previous concepts
- **Complete labs**: Hands-on practice is essential
- **Take quizzes**: Check your understanding regularly
- **Use the chatbot**: Ask questions using the embedded RAG chatbot

### For Instructors

- Follow the weekly breakdown for course planning
- Use the capstone guide for final projects
- Refer to the hardware tracks appendix for lab setup options

## Lab Setup Requirements

Before starting, ensure you have:

- **Operating System**: Ubuntu 22.04 (recommended)
- **ROS 2**: Humble or Iron distribution
- **Gazebo**: For simulation
- **Isaac Sim**: NVIDIA Isaac Sim (for advanced chapters)
- **Hardware**: See Appendix for on-premise vs cloud options

## Interactive Features

### RAG Chatbot

This book includes an embedded chatbot that can:

- **Ask the book**: Get answers from the full textbook corpus
- **Ask selected text**: Highlight text and ask questions about only that selection

The chatbot provides citations so you can verify information and dive deeper into specific sections.

## Structure of Each Chapter

Every chapter includes:

1. **Learning Objectives**: What you'll learn
2. **Core Concepts**: Key theoretical knowledge
3. **Hands-On Lab**: Practical exercises
4. **Checkpoint Quiz**: Self-assessment
5. **References**: Further reading

## Getting Help

- Use the chatbot for quick questions
- Check the glossary for definitions
- Review the hardware appendix for setup issues
- Consult the capstone guide for project help

## Safety & Ethics

Throughout this book, you'll find safety reminders and ethics considerations. Physical AI involves working with real robots and systems that can cause harm if misused. Always:

- Follow safety protocols
- Test in simulation first
- Understand ethical implications
- Respect privacy and data protection
